# 🚀 BrainyBeam MERN - React Day 03

A React practice project built during the BrainyBeam MERN Training. This project focuses on designing a modern homepage, creating Login and Register pages, handling forms using React, implementing two-way data binding, and managing form state with React Hooks.

## 📚 Topics Covered

- ⚛️ React Functional Components
- 🎣 React Hooks (`useState`)
- 🏠 Homepage UI Design
- 🔐 Login Page Design
- 📝 Register Page Design
- 📋 Form Handling in React
- 🔄 Two-Way Data Binding
- 🧠 Managing Form State with `useState`
- 🖊️ Handling Input Fields
- 📂 Organized Project Structure (`components` & `pages`)
