# 🚀 BrainyBeam MERN - React Day 04

A React practice project built during the BrainyBeam MERN Training. This project focuses on fetching data from APIs using Axios, managing state with React Hooks, creating reusable components, and building a responsive user interface with Bootstrap 5.

## 🌐 Live Demo

👉 **Live Link:** https://brainybeam-react-day-04.vercel.app/


---

## 📚 Topics Covered

- ⚛️ React Functional Components
- 🎣 React Hooks (`useState` & `useEffect`)
- 🌐 API Integration with Axios
- 📦 Fetching Data from REST APIs
- 🔄 Asynchronous Requests using `async/await`
- ⚠️ Error Handling with `try...catch`
- 📋 Rendering API Data using the `map()` Method
- ♻️ Reusable Card Components
- 🎨 Responsive UI with Bootstrap 5
- 📍 Google Maps Integration
- 📝 Responsive Contact Form
- 🏗️ Bootstrap Grid System
- 📂 Organized Project Structure (`components` & `pages`)
